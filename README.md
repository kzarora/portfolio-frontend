# ⚡ NOVA — Crazy React Frontend Template

A bold, animated portfolio/landing starter built with **Vite + React + Tailwind v4 + Framer Motion**.

## ✨ Features

- 🎨 Animated gradient blob background + grid overlay
- 🖱️ Custom magnetic cursor with hover detection
- 📊 Scroll progress bar
- 🪟 Glassmorphism UI components
- 🎞️ Framer Motion entrance & scroll animations
- ♾️ Infinite tech-stack marquee
- 💎 Fully responsive

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Then open the printed local URL (default http://localhost:5173).

## 🛠️ Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start dev server         |
| `npm run build`   | Production build         |
| `npm run preview` | Preview production build |

## 📁 Structure

```
src/
  App.jsx              # Layout + scroll progress
  main.jsx             # Entry point
  index.css            # Tailwind theme + effects
  components/
    Background.jsx     # Animated blobs + grid
    Cursor.jsx         # Custom cursor
    Navbar.jsx         # Glass navbar
    Hero.jsx           # Hero section
    Marquee.jsx        # Tech stack marquee
    Projects.jsx       # Project cards
    Contact.jsx        # CTA + footer
```

Customize colors in the `@theme` block inside `src/index.css`. Enjoy! 🔥
